## 🧾 **Student Finance Tracker**

### 🌍 Overview

The **Student Finance Tracker** is a web-based application built using **HTML, CSS, and JavaScript** to help students **manage their personal finances** more effectively.
It allows users to track income and expenses, monitor their spending trends, set budget goals, and stay financially disciplined — all through an accessible, easy-to-use interface.

---

### 💡 **Key Features**

* 🧮 **Dashboard Overview:** Displays total income, total expenses, and remaining balance.
* 💸 **Transactions Management:** Add, view, and sort all transactions by date, category, or amount.
* 🧠 **Savings Tip of the Day:** A motivational message to encourage smarter financial habits.
* 📊 **Weekly Trend Visualization:** View a quick summary of spending trends from the past 7 days.
* ⚙️ **Budget Settings:** Set a monthly spending cap and receive updates on progress.
* 💾 **Data Import & Export:** Easily back up or restore your financial data in JSON format.
* 🌈 **Responsive Design:** Optimized for both desktop and mobile screens.
* 🔍 **Regex Search:** Search your transaction records using powerful regular expressions.

### 🚀 **How to Run the Project**

1. **Clone or download** this repository:

   ```bash
   git clone https://github.com/<your-username>/student-finance-tracker.git
   ```
2. **Open** the `index.html` file in your preferred browser.
3. The app will load locally — no server setup is required.

> 💡 *Tip:* For best performance, open via a local development server (like VS Code’s “Live Server” extension).

---

### 🧭 **How to Use**

1. **Dashboard:**

    * View your total income, expenses, balance, and top spending category.
2. **Transactions:**

    * Add new transactions via the **Form** tab.
    * Use the search bar or regex patterns to filter records.
3. **Settings:**

    * Define your monthly budget limit and preferred currency.
    * Import or export your data as JSON files.
4. **About:**

    * Learn about the developer and project purpose.
